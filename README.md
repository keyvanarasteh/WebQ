# WebQ

**Cross-platform Web Intelligence and Cybersecurity Platform**

WebQ is a powerful, desktop-native intelligence gathering and reconnaissance tool built with Svelte 5, Tauri v2, and Rust (`web-analyzer`). It tracks WHOIS root registries, performs highly concurrent HTTP/HTTPS probing across subdomains, and validates security infrastructure.

## Features
- **Monolithic Intelligence Dashboards**: Tabbed analysis arrays utilizing Glassmorphism design tokens.
- **Deep Recon Infrastructure**: Active HTTP checking, fallback probing, array mapping.
- **Zero-Day Matrices**: Common port definitions, status codes, and headers matrices natively.
